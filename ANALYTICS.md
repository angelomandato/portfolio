## Blubrry Podcasting

### Blubrry Podcasting Statistics

Blubrry Podcasting Statistics is a podcast statistics application and service that provided detailed analytics for podcast episodes. It was developed in PHP and utilized a MySQL database to store and retrieve data. The application provided insights into podcast performance, including listener demographics, episode popularity, and download trends.

Blubrry Podcasting Statistics was capable of tracking podcast episodes hosted on Blubrry Podcasting (hosted logs) and other hosting providers (redirect logs). Hosted logs provided detailed information about downloads directly from Blubrry's servers which allowed for reporting how much of each episode was downloaded or played. Redirect logs allowed tracking of downloads from other hosting providers by redirecting requests through Blubrry's servers.

### Blubrry Podcasting Surveys

The Blubrry Podcasting Surveys was a unique survey system that allowed web visitors to fill out a simple survey to support podcast statistics data. The surveys were designed to gather information about podcast listeners, such as demographics, listening habits, and preferences. The survey data was then used to enhance the analytics provided by Blubrry Podcasting Statistics.

## Painless Analytics

### Datacenter and Cloud Service Lists

A GitHub auto-updating repository, GitHub actions run daily to maintain a list of datacenters and Cloud service providers.

Link: [https://github.com/painlessanalytics/datacenter-cloud-ip-lists](https://github.com/painlessanalytics/datacenter-cloud-ip-lists)

### Update Geo IPs Serverless Application

An AWS Lambda function serverless application written in Golang that maintains a Maxmind database of over 100 million datacenters, cloud service providers, and VPNs.

### Logger Server

A custom built Nginx server with built-in server side JavaScript. Server designed to efficiently log web beacon events and removes privacy information that would otherwise be logged. Logger Server can be hosted on AWS instances such as EC2 or ECS/EKS.

### Painless Analytics Tracker

An IIFE page event tracking privacy focused JavaScript, measures user engagement.
