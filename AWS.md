# Amazon Web Services experience for Angelo Mandato

Angelo Mandato has extensive experience with Amazon Web Services (AWS) starting in 2007.

## Angelo's History with AWS

Angelo Mandato started using AWS in summer of 2007 when developing podcast hosting service for Blubrry Podcasting. Amazon's Simple Storage service (S3) solved the need for an endless storage solution for large podcast media files.

In 2009 Angelo looked at AWS again to solve scaling issues Blubrry Podcasting was having by utilizing their compute service EC2. Not only did AWS service solve scaling issues, it also cut the monthly expense of hosting the podcast websites in half. Within 1 year most of Blubrry Podcasting's services were migrated to AWS, and by 2012 all web servers were hosted on AWS.

After the migration, referred to as a "lift and shift", Blubrry Podcasting's applications were slowly refactored and redeveloped to better utilize the AWS eco-system. The re-architecting included code modifications to leverage cloud-native features for long-term scalability and efficiency. Changes included moving file storage to S3, migrating databases to RDS MySQL / Aurora, and utilizing SNS/SQS services.

By 2015 Angelo was convinced that **Software Developers**, developers who focused on a specific programming language, would be replaced by **Software Architects** who focused on architecting applications for specific cloud service. To help transition the high tech community in Columbus he started the AWS Columbus User Group.

In 2016 Angelo attended his first re:Invent conference where he learend how to develop serverless Lambda functions in Python for the new Alexa smart devices, which started his path to learning serverless development.

Though his early work at Blubrry Podcasting utilized bash scripts in order to restore and scale infrastructure, by 2018 Angelo started using CloudFormation and other tools to maintain infrastrucutre on AWS.

In 2018 Angelo particpated with fellow Midwest AWS User Group leaders to organize thhe annual [AWS Midwest Community Day](https://www.midwestcommunityday.com) conference in Chicago. Angelo was instramental in hosting this conference to Columbus, Ohio in 2022 and 2024.

From 2007-2022 Angelo architected and oversaw the development of additional applications deployed on AWS at Blubrry Podcasting.

In 2022 Angelo moved on from Blubrry Podcasting and started consulting at [LeadingEDJE](https://www.leadingedje.com) helping clients build and enhance applications hosted on AWS.

In 2023 Angelo founded the startup Painless Analytics, which was part of the first cohort of startup businesses to particiopate in the 2023 AWS Build Accelerator startup program.
