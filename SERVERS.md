# Servers and Server Software experience for Angelo Mandato

Angelo has experience with a number of servers and server software, including web servers, email servers, and application servers. Angelo has developed applications and services that run on these servers, as well as contributed to the development and maintenance of server software.

## Web Servers

* Apache Web Server
* Nginx Web Server
* Xitami Web Server
* Microsoft IIS Web Server
* Lighttpd (Lighty) Web Server

Angelo has also developed modules and extensions for Nginx and Lighttpd web servers.

Angelo contributed code for 302 redirects with the mod_redirect Lighttpd module. https://redmine.lighttpd.net/issues/1446 Though the code was not used, it did lead to a different parameter “url.redirect-code” to provide the same functionality when value set to 302.
